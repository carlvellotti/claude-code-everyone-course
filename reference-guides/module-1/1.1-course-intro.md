# Введение в курс

## Добро пожаловать!

В этом курсе вы научитесь эффективно использовать OpenCode — AI-ассистента для разработки в терминале.

## Цели обучения

После прохождения курса вы сможете:

1. Уверенно работать с OpenCode в терминале
2. Использовать AI для ускорения разработки
3. Настроить память проекта с AGENTS.md
4. Работать с Git и GitHub через OpenCode
5. Создавать и развертывать проекты с помощью AI
6. Настраивать пользовательские команды и агентов

## Пререквизиты

- Базовые знания терминала/командной строки
- Понимание основ программирования (любой язык)
- Git (базовое использование)

## Формат курса

### Модуль 0: Установка и настройка
- Установка OpenCode
- Настройка провайдеров моделей
- Первый запуск

### Модуль 1: Основы работы
- Рабочее пространство OpenCode
- Работа с файлами и команды
- Использование агентов
- Настройка памяти проекта

### Модуль 2: Практический проект
- Планирование проекта
- Итеративная разработка
- Интеграция с GitHub
- Развёртывание

## Как проходить курс

1. **Выполняйте упражнения** — практика важнее теории
2. **Экспериментируйте** — пробуйте свои запросы
3. **Читайте AGENTS.md** — понимайте, как работает память
4. **Используйте Tab** — переключайте между build и plan

## Отличия OpenCode от аналогов

| Функция | OpenCode | Другие |
|---------|----------|--------|
| Модели | 75+ провайдеров | Ограниченный выбор |
| Цена | Бесплатные модели | Подписка $20+/мес |
| LSP | Встроенный | Нет |
| Агенты | build/plan | Нет |
| Приложение | CLI + Desktop | Только CLI |
| Конфигурация | ~/.opencode.json | .env файлы |

## Терминология курса

- **OpenCode** — AI-ассистент в терминале
- **AGENTS.md** — файл инструкций и памяти проекта
- **Провайдер** — сервис AI-моделей (OpenAI, Anthropic и т.д.)
- **Агент** — режим работы (build или plan)
- **MCP** — Model Context Protocol для расширений
- **LSP** — Language Server Protocol для анализа кода
