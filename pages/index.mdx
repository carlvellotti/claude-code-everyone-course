## Who This Course Is For

**Non-technical people** who want to use AI effectively for real work.

- No coding experience required
- No terminal experience required
- Just curiosity and a willingness to try something new

If you can have a conversation, you can use Claude Code.

## What You'll Learn

### [Module 1: Fundamentals](/fundamentals) (~3 hours)

- **File operations:** Read, transform, and create documents with the @ symbol
- **Visual workspace:** Set up a split-screen workflow to see files as Claude works
- **Parallel agents:** Launch multiple Claude instances to process files simultaneously
- **Custom sub-agents:** Create permanent AI advisors with different perspectives
- **Project memory:** Use CLAUDE.md so Claude remembers your project context
- **Commands & shortcuts:** Slash commands, keyboard shortcuts, power user features

### [Module 2: Vibe Coding](/vibe-coding) (~1-2 hours)

- **Setup & mindset:** The vibecoding approach - you describe, Claude builds
- **Planning:** Interview process to define clear requirements before building
- **Build & iterate:** Scaffold your app and refine it with screenshots
- **Version control:** Back up your work with GitHub
- **Go live:** Deploy to Vercel with a real URL anyone can visit

## How It Works

1. **Install Claude Code** - 15 minutes, one command
2. **Download course materials** - Includes practice files and lesson scripts
3. **Type `/start-1-1`** - Claude begins teaching you interactively
4. **Learn by doing** - Every concept includes hands-on practice

These reference pages provide deeper explanations and quick lookups while you work through the interactive lessons.
